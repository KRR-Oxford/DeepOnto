deeponto.utils.tree module

A tree implementation for ranges (without partial overlap). - parent node’s range fully covers child node’s range, e.g., [1, 10] isParentOf [2, 5] - partial overlap between ranges not allowed, e.g., [2, 4] and [3, 5] cannot appear in the same RangeNodeTree - non-overlap ranges are on different branches - child nodes are ordered according to their relative positions
